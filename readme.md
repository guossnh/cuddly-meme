#   这是一些淘宝使用的脚本的合集   主要通过油猴子实现

##  sell_list 
    这是淘宝已售出物品列表的增强功能  现在主要实现了宝贝区分  自己专用
##  sell_detail_copy 
    这是淘宝已售出物品详情页的增强功能, 支持 一键复制  支持  chrome43+ ff41+ 版本的浏览器  主要是因为 剪贴板的赋值有问题
##  sell_all_add 
    这是统计当页的产品的种类数目 并且 计算 当页的当前产品的 计算和.(会计用的 ,其他人没吊用)
